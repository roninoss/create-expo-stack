Tamagui is a cross-platform UI kit that helps you ship consistent experiences across React Native and web.

## Key Features

- **Cross-platform components:** Build once for native and web with shared tokens.
- **Token-driven theming:** Use a single config to define your design system.
- **Composable primitives:** Start with layout primitives like stacks and buttons.

## Notes

When Tamagui is selected, Create Expo Stack adds a starter `tamagui.config.ts` and wraps the app in `TamaguiProvider`.
