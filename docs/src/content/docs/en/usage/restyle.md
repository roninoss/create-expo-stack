Restyle is a theming system from Shopify that helps you define design tokens and apply them consistently across components.

## Key Features

- **Design tokens:** Define colors, spacing, typography, and breakpoints in one place.
- **Typed styles:** Generated theme types keep styling safe and predictable.
- **Composable components:** Build reusable UI primitives with shared variants.

## Notes

When Restyle is selected, Create Expo Stack adds a starter theme and wraps the app in a `ThemeProvider`.
